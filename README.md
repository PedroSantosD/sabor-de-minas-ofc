# 📌 Sobre o Projeto

O **Sabor de Minas** é um sistema de delivery moderno desenvolvido para uma empresa real.

O projeto possui:

* 🛒 Cardápio online
* 📱 Responsividade mobile
* 🔐 Login admin
* 📊 Dashboard administrativo
* 📈 Gráficos de vendas
* 🖼️ Upload de imagens
* 📦 Gerenciamento de produtos
* 📲 Integração com WhatsApp
* ☁️ Banco de dados com Supabase
* 🚀 Deploy online via Netlify
* 📲 PWA para uso no celular
